## Bank_streaming

Projekt przygotowany do uruchomienia przez `docker compose up -d --build`.

## Start

```bash
docker compose down --volumes --remove-orphans
docker compose up -d --build
```

## VS Code

Wejdź na `http://127.0.0.1:8080`.

## Jupyter

Notebooki uruchamiaj w code-server po wejściu do folderu `/home/coder/workspace`.
